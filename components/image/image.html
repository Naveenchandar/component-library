Image

Images can be responsive to fit the parent's width, and also can be customised to be round shaped

Squared Image

Square Images are the square shaped, with a default dimension of 90px X 90px.


Responsive Image

Responsive Images fit the width of container. It's height will get adjusted by keeping the aspect ratio same. If you want to change the aspect ratio, you will have to crop the image.


Rounded Image

You can add class img-rounded to make your image round shaped.
